# 📚 高中 AI 學習陪伴 · tw-senior-high-ai-tutor

> 讓每個孩子學習時，都能享受「高手名師」教法——用 AI 把各科主題講得有趣、好懂、好記，並自動整理成可複習的知識庫。

輸入一個主題，AI 會依各科教學法原型（故事、口訣、圖解、生活實證…）產生**生活連結與記憶法**，標出**考試重點**，並寫成結構化筆記、串連跨科知識，幫助你**預習、複習、補強**。

---

## ⚠️ 使用前必讀
- **內容由 AI 生成，可能有錯**：一切以課本與大考中心公告為準。每篇筆記都有「信心度」與「待確認項」，看到要查證的請務必查證。
- **未成年請由家長／老師陪同**：本工具需搭配 Anthropic Claude Code，其服務有年齡限制。
- **需付費**：專案本身免費開源，但執行 AI 會產生 Claude Code 的使用費用。
- 完整說明請讀 **[免責聲明](免責聲明.md)**。

---

## 🚀 怎麼開始
第一次需安裝設定，之後每次只要一行指令。完整零基礎教學見 **[快速開始](docs/快速開始.md)**。

設定好後，在 Claude Code 裡輸入：
```
/learn <科目> <主題>
```
例如 `/learn 數學 矩陣運算`、`/learn 歷史 戊戌變法`。AI 會自動完成「生成教學 → 寫成筆記入庫 → 查證標註信心度 → 建立跨科連結」，最後告訴你筆記存在哪、需不需要再確認。

**支援科目**：國文、英文、數學、物理、化學、生物、地球科學、歷史、地理、公民與社會。

複習時直接打開對應科目資料夾的筆記即可，每篇都有「一句話總結 / 考試重點 / 易錯點 / 一分鐘自我檢測」。

> 💡 本專案是一個 **Obsidian 知識庫（vault）**。用 [Obsidian](https://obsidian.md) 開啟此資料夾，跨科 `[[連結]]`、關係圖譜與 Mermaid 圖才能正常呈現；純文字編輯器只看得到原始 Markdown。
> 關聯圖（Graph view）已內建篩選，定位為**學習路線圖**：顯示各科筆記、各科主題地圖（MOC）中樞，以及**尚未建立的主題**（暗淡的幽靈節點）——讓孩子看出主題如何關聯、下一步往哪加廣加深；只隱藏系統檔與使用說明。
> 🧭 **怎麼用來導航**：看到想學的暗淡主題，就點它旁邊**金色（全庫總圖）或橘色（各科 MOC）的中樞節點**，進入該科主題地圖——裡面有完整待學清單與 `/learn <科目> <主題>` 生成指令。（Obsidian 無法讓你直接點幽靈節點跳轉，所以改點發亮的中樞。）想只看已建筆記，可在圖內清掉篩選字串。

## 🧠 它怎麼運作
兩組 AI Agent 接力合作：

```
你輸入主題
   │
   ▼
科目教學團隊 Agent ──► 筆記轉換 Agent（套模板入庫）
（每科 3 種教學法原型，      │
  接力：興趣→理解→記憶）      ▼
                        內容檢查 Agent（查證、標信心度）
                             │
                             ▼
                        知識串連 Agent（建跨科連結）
```

- **教學團隊**：10 科、每科 3 個去識別化的「教學法原型」角色。團隊定義見 [`_system/agents/科目Agent團隊/_總覽.md`](_system/agents/科目Agent團隊/_總覽.md)。
- **執行版 Agent**：在 [`.claude/agents/`](.claude/agents/)，由 [`/learn`](.claude/commands/learn.md) 指令串起。
- **設計原則**：① 生成與品控分離 ② 不模仿實名教師、只萃取教學法 ③ AI 不取代人工審核（保留信心度與待確認項）④ 逼孩子先思考（自我檢測）。

## ✅ 我們怎麼面對「AI 會講錯」
這是本專案最在意的事：
- **內容檢查 Agent 會實際上網查證**公式、年代、課綱範圍等硬事實，而非憑記憶。
- 每篇筆記標 `信心度`（高/中/低）與 `檢查狀態`；不確定的列「待確認項」交人工複核——**我們選擇誠實，而非假裝全對**。
