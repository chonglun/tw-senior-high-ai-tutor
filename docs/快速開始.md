# 🚀 快速開始（零基礎照著做）

第一次用會花一點時間設定，**設定好之後，每次學習只要打一行指令**。
> 👨‍👩‍👧 未滿法定年齡的同學，請和家長或老師一起完成設定。請先讀 [免責聲明](../免責聲明.md)。

---

## 💰 先了解費用（很重要）
- 本專案**免費開源**，但它要靠 Anthropic 的 **Claude Code** 才能運作。
- **Claude Code 是付費服務**（可用訂閱方案，或 API 按量計費）。
- 每次 `/learn` 會呼叫多個 AI 步驟，會消耗額度。建議：
  - 由家長／老師管理帳號與付費。
  - 一開始先少量使用，觀察花費，再決定用量。
  - 留意 Claude Code 顯示的用量資訊。
- 最新價格與方案以 [Anthropic 官網](https://www.anthropic.com) 為準。

## 步驟一：安裝 Claude Code
1. 前往官方安裝說明：https://docs.claude.com/claude-code
2. 依你的系統（Windows / macOS）照官方步驟安裝。
3. 安裝過程需要一個 Anthropic 帳號並登入授權（**由家長／老師建立**）。

## 步驟二：取得本專案
**方法 A（推薦，會用 git）：**
```
git clone https://github.com/chonglun/tw-senior-high-ai-tutor.git
```
**方法 B（不會 git）：** 在 GitHub 頁面點綠色「Code」→「Download ZIP」，下載後解壓縮到你記得的資料夾。

## 步驟三：初始化版控與品質守門（建議）

> 💡 這個步驟讓你的筆記庫有「存檔紀錄」，並在你儲存前自動提醒你完成 AI 查證，確保筆記品質。

在終端機進入專案資料夾，執行一次安裝腳本：
```
cd tw-senior-high-ai-tutor   # 或你解壓縮後的資料夾名稱
sh scripts/install-hooks.sh
```
看到「✅ Pre-commit hook 安裝完成！」就成功了。**這個步驟只需要做一次。**

之後每次你執行 `git commit` 存檔，系統會自動確認筆記品質沒問題才讓你儲存；如果有筆記還沒完成查證，它會告訴你要在 Claude Code 裡跑哪個步驟。

## 步驟四：用 Claude Code 打開這個資料夾
在終端機（Terminal）進到專案資料夾，再啟動 Claude Code：
```
cd 你存放專案的資料夾
claude
```

## 步驟五：開始學習！
在 Claude Code 裡輸入：
```
/learn 數學 矩陣運算
```
格式是 `/learn <科目> <主題>`。AI 會自動：
1. 用該科的教學團隊產生內容
2. 寫成一篇筆記存到對應科目資料夾
3. 查證內容、標註信心度
4. 建立和其他主題的連結

完成後它會告訴你筆記存在哪、信心度多少、哪些地方還要再確認。

## 支援的科目
國文、英文、數學、物理、化學、生物、地球科學、歷史、地理、公民與社會

## 怎麼複習？
直接打開對應科目資料夾裡的 `.md` 筆記檔即可。每篇都有「一句話總結、考試重點、易錯點、一分鐘自我檢測」。

> 💡 **建議用 [Obsidian](https://obsidian.md) 開啟整個資料夾**（免費）。筆記之間用 `[[連結]]` 互通、附 Mermaid 圖與關係圖譜，在 Obsidian 裡才看得到完整效果；用一般文字編輯器只會看到原始碼。每科資料夾的 `<科目>MOC.md` 是該科的主題地圖入口。

## ❓ 常見問題
**Q：輸入 `/learn` 卻說找不到指令？**
A：Claude Code 在啟動時才載入專案的指令與 agent。請**關掉再重新啟動** `claude`，並確認你是在專案資料夾裡啟動的。

**Q：它要我確認權限？**
A：本專案已附共用權限設定（`.claude/settings.json`），正常情況不太會一直問。若仍出現，確認後同意即可。

**Q：內容會不會有錯？**
A：會有可能。請務必看筆記裡的「信心度」與「待確認項」，並以課本和大考中心為準。詳見 [免責聲明](../免責聲明.md)。

**Q：可以學課綱以外、更深更廣的內容嗎？**
A：可以。`/learn` 任何主題都行，但範圍外的內容更要自己查證。
