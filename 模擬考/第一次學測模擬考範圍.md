## 🔤 [[英文MOC]]（第 1 冊 ～ 第 2 冊）
- [ ] **核心單字**：熟背高一範圍（Level 1~3 或 Level 4 部分）單字，特別注意一字多義與搭配詞 (collocation)。

- [ ] **克漏字與文意選填**：透過歷屆試題熟悉上下文推論技巧。

## 📐 [[數學MOC]]（第 1 冊，數A、數B 不分卷）
- [ ] **數與式**：絕對值方程式與不等式、乘法公式、算幾不等式。
- [ ] **直線與圓**：直線方程式、點到直線距離公式、圓的方程式、圓與直線的關係。
- [ ] **多項式函數**：一次與二次函數（配方法、頂點、極值）、多項式的除法原理、餘式與因式定理。
- [ ] **指數與對數**：指數律、對數的定義與運算性質、科學記號與常用對數。

## 🔬 自然科（物理、化學、生物、地科 上半冊）
### 物理（全，上半冊，參見 [[物理MOC]]）
- [ ] **科學的態度與方法**：物理量的單位（SI 制）、因次分析。
- [ ] **物質的組成與交互作用**：原子模型、四大基本作用力（重力、電磁力、強核力、弱核力）。
- [ ] **物體的運動**：等速度運動、等加速度運動（v-t 圖判讀與三大公式）、自由落體。

### 化學（全，上半冊，參見 [[化學MOC]]）
- [ ] **物質的分類與組成**：純物質與混合物、物理與化學變化。
- [ ] **物質的構造**：原子結構（質子、中子、電子）、同位素、八隅體規則。
- [ ] **化學式與化學計量**：莫耳數計算、限量試劑、濃度換算（體積莫耳濃度、重量百分率濃度）。

### 生物（全，上半冊，參見 [[生物MOC]]）
- [ ] **細胞的構造與功能**：動植物細胞胞器比較（粒線體、葉綠體、核糖體等）、細胞膜的運輸機制。
- [ ] **細胞的能量代謝**：光合作用（光反應、碳反應）與呼吸作用（有氧、無氧）的基本概念。
- [ ] **生殖與遺傳**：細胞分裂與減數分裂的差異、孟德爾遺傳法則（分離律、獨立分配律）。

### 地球科學（全，上半冊，參見 [[地球科學MOC]]）
- [ ] **地球的故事**：地球的起源與演化、相對地質年代與絕對地質年代（半衰期）。
- [ ] **從地球看宇宙**：星系與宇宙膨脹、恆星的視星等與絕對星等。
- [ ] **千變萬化的大氣**：大氣的分層（對流層、平流層等）、溫室效應原理。

## 🌍 社會科（歷史、地理、公民與社會）
### 歷史（第 1 冊：臺灣史，參見 [[歷史MOC]]）
- [ ] **多元族群社會的形成**：原住民族、大航海時代的荷西時期、清領時期的移民與社會。
- [ ] **現代國家的形塑**：日本統治時期的現代化建設、政治與社會運動。
- [ ] **經濟與文化的多樣性**：戰後台灣的政治經濟發展與民主化歷程。

### 地理（第 1 冊：地理基礎，參見 [[地理MOC]]）
- [ ] **地理技能與方法**：經緯度、比例尺、投影法概念。
- [ ] **地理資訊系統 (GIS)**：向量與網格資料、空間疊圖分析。
- [ ] **氣候系統**：大氣環流、行星風系、台灣與全球氣候類型判斷。
- [ ] **地形系統**：河流地形、海岸地形、火山地形、風化與冰河地形的成因與特徵。

### 公民與社會（第 1 冊：社會與政治，參見 [[公民與社會MOC]]）
- [ ] **公民身分**：人權的發展與基本人權保障。
- [ ] **國家與認同**：國家的構成要素、民主國家的基本原則。
- [ ] **民主治理**：政府體制（內閣制、總統制、雙首長制）、我國中央政府的職權與五權分立。
- [ ] **公共意見與參與**：選舉制度（單一選區兩票制等）、政黨與利益團體的差異。
