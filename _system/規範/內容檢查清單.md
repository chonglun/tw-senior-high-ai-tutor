# 內容檢查清單

內容檢查 Agent 與人工複核共用。逐項打勾。

## A. 硬事實（一律保守，可疑即標待人工複核）
- [ ] 公式是否正確、符號定義清楚？
- [ ] 定義、定理、定律的敘述是否精確？
- [ ] 年代、人名、地名、事件（歷史/地理）是否正確？
- [ ] 數據、單位、數量級是否正確？
- [ ] 是否引用了已過時或已修訂的內容？
- [ ] 「為什麼要學（Why）」宣稱的應用／價值／好處是否真實、站得住腳，**不誇大、不杜撰**（如「學了 X 就能做 Y」需屬實）？

## B. 生活連結與比喻
- [ ] 比喻是否科學正確、不建立錯誤直覺？
- [ ] 例子是否貼近高中生生活、易懂？

## C. 課綱與考試
- [ ] 主題是否在 108 課綱學測範圍內？
- [ ] 冊別 / 難度 / 考試頻率標註是否合理？
- [ ] 「必背」「常考」是否真的高頻，不誤導複習方向？

## D. 結構與連結
- [ ] 模板區塊是否齊全、順序正確？
- [ ] 💡「為什麼要學（Start with Why）」是否存在、置於開頭、具體且能引發動機（非空泛口號）？
- [ ] 自我檢測 ≥3 題，且答案與內文一致？
- [ ] 跨科連結是否相關且格式正確？

## E. 用詞精確度與錯字（逐字校對，發現必明確指出「錯字 → 正確字」）
- [ ] 錯別字、音近/形近誤用：如「在/再」「的/得/地」「因該→應該」「藉由/籍由」「決對→絕對」「凝聚/凝結」。
- [ ] 專有名詞、術語用字與課本一致：如「亞佛加厥數」「軟流圈」「閣揆」「氧化還原」「韻文/韻文」「鋒面」（勿寫成「峰面」）。
- [ ] 用詞精確、不似是而非：常見混淆——速率 vs 速度、重量 vs 質量、熔化 vs 溶解、體積 vs 容積、權利 vs 權力、效率 vs 效益、增加 vs 成長。
- [ ] 標點、全形/半形、單位符號正確：°C、mol、m/s²、% 等；中英數之間空格一致。
- [ ] 有無亂碼、缺字或生成殘留符號（如 �、重複字、半截詞）。
- [ ] 文句是否通順、無語病或殘缺句。

- [ ] 圖選用的類型是否恰當（流程→flowchart、狀態變化→stateDiagram、時序/互動→sequenceDiagram、層級分類→mindmap、比例→pie、時間軸→timeline/gantt）？
